# WPF_HCI: Email Example

This repository contains the development of a Human-Computer Interaction (HCI) project implemented using Windows Presentation Foundation (WPF) and C#.

The main objective is to explore and apply principles of Usability and User Experience (UX) to create a robust, intuitive, and efficient desktop application.

## Technologies Used

* **Windows Presentation Foundation (WPF):** Framework used for building the user interface.
* **C#:** Main development language.

## Installation and Running

To clone and run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Gpascual11/WPF_HCI.git]
    ```
2.  **Open the Project:**
    Open the solution (`.sln`) with Visual Studio
3.  **Restore NuGet Packages:**
    Ensure all necessary NuGet packages are automatically restored. If not, right-click the solution and select `Restore NuGet Packages`.
4.  **Run:**
    Press `F5` or the 'Start' button in Visual Studio.
